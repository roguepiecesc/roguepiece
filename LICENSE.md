## 🛠️ Troubleshooting Common Issues

- ❌ **Script GUI Does Not Appear**
  The overlay may be blocked by your Roblox window settings. Try running in windowed mode instead of fullscreen. Make sure the **Rogue Piece Script** file was fully extracted and not corrupted during download.
  **Solution:** Re-download the script from the official badge link, extract to `C:\Desktop\RogueScript`, and relaunch both Roblox and the loader.

- ❌ **Rogue Piece Script Auto Fish Stops Working**
  This usually happens after a game update changes the fishing animation timing. The Rogue Piece Script Auto Fish module needs to match the current game version.
  **Solution:** Update to the latest **Rogue Piece Script** version and restart. If the issue persists, set `AUTO_FISH_REEL_SPEED` to `"normal"` in your config.

- ❌ **Auto Dungeon Gets Stuck on a Specific Floor**
  Some dungeon floors have unique boss mechanics that require updated logic in the Rogue Piece Script Dungeon module.
  **Solution:** Update the script and set `DUNGEON_HEAL_THRESHOLD` to `40` for tougher floors. The Best Rogue Piece Script will handle the rest.

- ❌ **Lag or Frame Drops While Running Script**
  Running too many modules at once can overload lower-end devices.
  **Solution:** Disable features you are not actively using in the Qol Rogue Piece Script config. Set `FARM_RADIUS` to `100` or lower to reduce processing load.

- ❌ **Server Hop Connects to Same Server**
  A known edge case when few servers are available during off-peak hours.
  **Solution:** Wait a few seconds and try again, or manually set a different region in the **Rogue Piece Script** advanced settings.
